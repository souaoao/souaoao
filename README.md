## Research

### Gait Assist Design using Musculoskeletal Simulation (Completed)
筋骨格シミュレーション（AnyBody）を用いて、  
歩行時の関節トルク低減を目的としたアシスト設計を行いました。

- 歩行データ解析（関節角度・関節トルク・床反力）
- アシストトルク波形の設計
- シミュレーションによる評価

この研究を通して、生体力学・ロボティクス・シミュレーション統合の理解を深めました。  
現在はこの経験を基に、より実応用に近いロボティクスおよびAIアプリ開発に取り組んでいます。
